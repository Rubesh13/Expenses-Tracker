# Expense Tracker

A simple Expense Tracker built using Python and MySQL. This project helps users manage and track their daily expenses through a command-line interface.

## Features
- Add expenses
- View all expenses
- Update expenses
- Delete expenses
- Store data in MySQL database

## Technologies Used
- Python
- MySQL
- mysql-connector-python

## Project Structure
```
Expense-Tracker/
│
├── main.py
├── db.py
├── database.sql
└── README.md
```
## Setup
1. Install Python
2. Install MySQL
3. Create the database using `database.sql`
4. Install required package:

```bash
pip install mysql-connector-python
```
5. Run:

```bash
python main.py
```
